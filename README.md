# アニ曜日 — このアニメ、サブスク配信は何曜日?

今クールにテレビ・ウェブで放送されるアニメの配信曜日を、動画配信ストリーミングサービスごとに確認できるウェブアプリです。情報は [Annict GraphQL API](https://developers.annict.com/docs/graphql-api/beta) から取得します。

## 機能

- 昨日の曜日を先頭にした 1 週間カレンダー(今日の列はハイライト表示)
- 各曜日にアニメタイトルと、配信しているストリーミングサービスの favicon を表示
- タイトルをクリックすると公式サイト(なければ Annict の作品ページ)を新しいタブで開く
- 既定では今クールの作品のみ表示。ヘッダーの ◀ ▶ で任意のクールに切り替え可能(今クール以外では日付・「今日」バッジは非表示)
- 表示するストリーミングサービスを複数選択でき、契約していないサービスを除外できる
- 週内で 2 番目以降の配信(遅れ配信)は畳んで表示。「遅れ配信を隠す」で完全非表示にもできる(選択中サービス内の最速配信は常に表示される)
- 視聴しない作品は × ボタンで非表示にでき、非表示リストから復元できる
- 気になる作品は ☆ ボタンでお気に入り登録すると、金色の枠と ★ で強調され各曜日の先頭に浮上する。お気に入りの遅れ配信は畳まれず展開表示される(「遅れ配信を隠す」を ON にした場合はお気に入りも含め最速配信のみ表示)
- テーマはヘッダーのボタンでシステム追従 / ライト / ダークを切り替え可能(システム追従では OS の設定変更にリアルタイムで反応)
- 対象メディアは TV / WEB のみ。テレビ局での放送予定は表示せず、ストリーミングサービス([src/services.ts](src/services.ts) の対応表に載っているもの)のみカレンダーに載せる

## セットアップ

```sh
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開きます。

### Annict アクセストークン

初回起動時にトークンの設定画面が表示されます。

1. [Annict の設定 → アプリケーション](https://annict.com/settings/apps) を開く
2. 「個人用アクセストークン」を作成(スコープは「読み込み専用」で OK)
3. トークンを貼り付けて保存(ブラウザの localStorage に保存されます)

`.env.local` に `VITE_ANNICT_TOKEN=...` を書いておくと、その値が初期値として使われます。

## 仕組み

- 今クール(日本時間で 1-3 月: 冬 / 4-6 月: 春 / 7-9 月: 夏 / 10-12 月: 秋)の作品を `searchWorks` で全件取得
- 各作品の放送・配信予定(`programs`)から、ストリーミングサービスのチャンネルのみ抽出
- サービスごとに「現在時刻に最も近い配信予定」の曜日・時刻(日本時間)を採用し、曜日ごとにまとめて表示
- Annict API へのリクエストは Vite の開発サーバーのプロキシ(`/graphql` → `https://api.annict.com`)経由で行い、CORS を回避

## ストリーミングサービスの追加

[src/services.ts](src/services.ts) の `SERVICES` 配列に `{ key, label, domain, keywords }` を追加してください。`keywords` は Annict のチャンネル名との部分一致で判定され、`domain` から favicon を取得します。
